 Super-Store-Sales-Dashboard
1. 📌 Project Overview
This project analyzes Super Store sales data using Power BI to provide insights into sales performance, profit trends, customer behavior, and regional performance. The dashboard helps stakeholders make data-driven decisions through interactive visualizations.
2. 🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing products and customers
Compare sales across categories and sub-categories
Monitor sales by region, state, and shipping mode
Track customer purchasing patterns
3 . 🛠️ Tools Used
Power BI
Power Query
DAX
Excel/CSV Dataset
4. 📈 Dashboard Features
KPI Cards (Total Sales, Profit, Quantity, Delivery Days)
Sales & Profit by State Map
Top 5 Products by Sales
Top 5 Customers by Profit
Sales by Category
Sales by Sub-Category
Sales by Segment
Sales by Payment Mode
Sales by Ship Mode
Interactive Filters (Region, Category, Month, Ship Mode)
5. 🔍 Key Insights
Total Sales: 1.57M
Total Profit: 175K
Total Quantity Sold: 22K
Consumer segment contributes the highest sales.
Office Supplies category generates the most revenue.
Standard Class is the most frequently used shipping mode.
